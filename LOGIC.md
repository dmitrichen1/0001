## Формула материала (ГОТОВО, проверено на кейсе 91 700 ≈ 91 900)
```
курс = max(ЦБ, планка) × 1.08   (планка: $85, €95; ЦБ берётся автоматически)
цена_листа_руб = цена_USD/EUR × курс
половинка = 0.5 × цена целого листа + 2000 руб (распил)
камень = (целые + половинки×0.5) × цена_листа_руб
с_наваром = (камень + распил) × 1.3333   (навар материала 25% изнутри)
итог = с_наваром + 6000 × кол-во_брендов (доставка)
округление вверх до 100 руб
цены берём ВСЕГДА без скидок
```

## Раскрой (логика зафиксирована, код — позже)
- Бот прикидывает кол-во листов для СТОИМОСТИ, финальную раскладку отдаёт мастеру
- Деталь = полоса шириной = глубина детали; по длине листа детали встык с зазором на рез
- Зазор на рез: беглый 10 мм, сложный/косой 20-30 мм, плотный 5 мм (диск 4 мм)
- Половинка листа = лист разрезанный по ширине (длина целая): Джамбо 3220×810, Нормал 3050×720
- Поворот детали можно, НО учитывать рисунок (направленный — стык с переходом рисунка)
- Длинная деталь (>длины листа) делится: не мелкими кусками, по возможности в одну ориентацию
- Цель: красивый раскрой при мин. материале. Но финал — мастер.
- Камень из наличия, минимальный по размеру и цене (не «уточнять наличие»)
- ВСЕГДА помечать «расчёт предварительный»

## Кромка
- По умолчанию: длина + ширина (когда непонятно)
- Остров → 4 стороны (периметр); ванная/тумба → 3 стороны; пристенная → длина + торцы
- Если геометрия явно даёт большую кромку — считать по большей

## Структура полной сметы (себестоимость + навар по каждой позиции)
1. Материал (камень по формуле, навар 25% уже внутри)
2. Доставка на производство (6000/бренд)
3. Замер (6000)
4. Цеховые работы (вырезы, полировка, профиль/кромка...)
5. Вспомогательные работы и материалы
6. Доставка на объект (7000 + км от МКАД ×250)
7. Монтаж (установка столешницы 3000/м.п. ...)

## Навар — ПО КАЖДОМУ ТИПУ ОТДЕЛЬНО (не общий на хвост!)
Причина: мелкие допы и подорожания не должны съедать общий интерес.
Черновая сетка для доработки (НЕ финал):
- камень: 33%
- доставка: 20%
- замер: 25%
- цех: 75%
- монтаж: 50%
Контроль прибыльности: цель 25%+, диапазон 25-50% хорошо, 15-25% пойдёт,
10-15% слабо, <10% — пауза на консультацию с директором, не отказывать сразу.

## Люфт цены (от якоря клиента)
- Проект ~150 тыс → люфт +5-10%
- Проект ~600 тыс → больше, но потолок +20% (выше — наглёж)
- При возражении: опускаемся, выясняем бюджет/цену конкурента, пересчитываем раскрой
- Честно по товару (не врать о свойствах), люфт только в цене
- Если клиент мог взять дорогой камень → аналог называем с люфтом вверх, смотрим реакцию

## Интерес посредника
- Дизайнер/строитель может попросить заложить свой интерес — уточнить 5-15%, добавить сверху
